🚀 Production-Ready Linux Server – Capstone Project

🎯 Goal: Deploy a secure, monitored web app with database + automation + backups.

⏱️ Duration: 2 weeks after month 1

📋 Requirements Checklist

  • Two VMs / EC2 instances (Web + DB)
  • SSH key-only & firewall rules
  • Ansible playbook for provisioning
  • Deploy Flask/PHP app that queries DB
  • Log rotation + monitoring script (CPU/disk)
  • Automated DB backups (cron)
  • Document restore procedure + troubleshooting
📌 Example: Backup script snippet
#!/bin/bash
mysqldump -u root mydb > /backup/db/mydb_$(date +%Y%m%d).sql
